- `PRIMARY KEY (id)`

## Entidad JPA

```java
@Entity
@Table(name = "productos")
@Getter
@Setter
public class Producto {
    @Id @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @NotBlank @Size(max = 150)
    private String nombre;

    @Size(max = 500)
    private String descripcion;

    @NotBlank @Size(max = 100)
    private String categoria;

    @NotNull @DecimalMin("0.0")
    @Column(name = "precio_base")
    private Double precioBase;

    private boolean activo = true;

    @Column(name = "fecha_creacion", updatable = false)
    private LocalDateTime fechaCreacion;

    @Column(name = "fecha_actualizacion")
    private LocalDateTime fechaActualizacion;
}
```

**Lifecycle Callbacks:**
- `@PrePersist` (método `beforeCreate()`): Asigna `fechaCreacion` y normaliza (trim) los campos de texto.
- `@PreUpdate` (método `beforeUpdate()`): Asigna `fechaActualizacion` y normaliza (trim) los campos de texto.

## Repositorio

### `ProductoRepository`

| Método | Descripción |
|---|---|
| `findByActivoTrue()` | Listar solo productos activos |
| `findByIdAndActivoTrue(Long)` | Buscar producto activo por ID |
| `existsByNombreIgnoreCase(String)` | Verificar unicidad de nombre |
| `existsByNombreIgnoreCaseAndIdNot(String, Long)` | Verificar unicidad excluyendo registro actual |

CREATE TABLE IF NOT EXISTS productos (

    precio_base DOUBLE NOT NULL,
    activo BOOLEAN NOT NULL DEFAULT TRUE,

);
